2.2 Compiling with qmake

Marsyas can be built with qmake, which is the Makefile generator in Qt.

Requirement: you must have Qt-4.2.2 or later. (no, 4.2.1 doesn't work. The documentation editor learned this the hard way.)

To build Marsyas, go to the base of the source tree and edit marsyasConfig.pri. Once you have selected the options you want (i.e. audio API, Ogg Vorbis support, etc), simply type 

     qmake
     make
     make install

2.1 Linux Installation

Marsyas is mainly developed under Linux so installing under Linux is quite straightforward and can be done using the standard autoconf installation steps where > is whatever command-line prompt you have on your system. 

     > ./configure
     > make
     > make install                 (as root user)

By default Marsyas assumes that the ALSA sound system and corresponding library and headers are installed. Most new Linux distribution are using ALSA. If you want to install Marsyas using the older OSS sound system you can do: 

     > ./configure --enable-oss

A frequent variation (if you don't have root priviledges) is to install Marsyas in your home directory (replace /home/gtzan with the appropriate path for your home directory). This can be accomplished by doing: 

     > ./configure --prefix=/home/gtzan
     > make
     > make install

In addition there are several configure options that can be used to enable/disable assertions, enable/disable debugging and include optional packages such as support for reading mp3s. See Configure Options.
